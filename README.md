# 🔢 Number Guessing Game

A fun, interactive, and beautifully designed **Number Guessing Game** built with HTML, CSS, and JavaScript. Challenge yourself to guess the secret number within the fewest attempts possible!

## 🚀 Features

- **Interactive Gameplay:** Simple and engaging user interface.
- **Dynamic Feedback:** Get real-time hints indicating if your guess is too high, too low, or correct.
- **Attempt Tracking:** Keeps count of how many attempts you've made.
- **Responsive Design:** Playable on desktops, tablets, and mobile devices.
- **Reset Functionality:** Easily restart the game without refreshing the page.

## 🎮 How to Play

1. The game will randomly select a secret number between a specified range (e.g., 1 to 100).
2. Enter your guess in the input field provided.
3. Click the **"Guess"** (or submit) button.
4. Read the feedback:
   - ⬆️ **Too High!** Try a smaller number.
   - ⬇️ **Too Low!** Try a larger number.
   - 🎉 **Correct!** You've guessed the secret number!
5. Hit the **"Play Again"** button to start a new round.
